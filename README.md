# DOOnjons et Dragons

**Le joueur incarne un ou plusieurs personnages (au choix) appartenant à différentes classes.  
L’objectif est de parcourir trois donjons remplis de monstres variés, à vaincre pour progresser dans l’aventure.**.

---

## ⚙️ Installation et exécution

1. Ouvrir le projet dans **IntelliJ IDEA** (ou tout autre IDE compatible Java).  
2. Vérifier que le **JDK** est correctement configuré (version **openjdk-23**).  
3. Lancer le fichier `Main.java` pour démarrer le jeu.

## Concepts clés

- Programmation orientée objet (héritage, polymorphisme, encapsulation)  
- Gestion d’inventaire et de statistiques  
- Architecture modulaire en Java

## Description du jeu


Au fil du jeu, le joueur peut :
- Ramasser et **équiper des objets** trouvés dans les donjons.  
- Gérer son équipement à l'aide d'un **inventaire**.  
- Profiter des **statistiques uniques** de chaque classe de personnage.  
- Utiliser des **sorts spécifiques**, disponibles pour certaines classes (ex. : le mage).
